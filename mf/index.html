❓ 生活常见问题✕

💬 生活常见问题

朋友欠钱不还怎么办？
先协商→保留借条/转账记录→诉讼时效3年（第188条）→可起诉。民间借贷利率上限受第680条约束。
借条没写利息还能要利息吗？
视为无利息（第680条）。但逾期不还可以主张逾期利息。
离婚冷静期是多久？
30天（第1077条）。协议离婚需双方亲自到婚姻登记机关申请，30天内任何一方可撤回。
高空抛物砸到人谁负责？
侵权人赔偿；找不到侵权人的，可能加害的建筑物使用人补偿（第1254条）。物业未尽安全保障义务也需担责。
买到假货怎么维权？
退一赔三（消费者权益保护法第55条）；食品问题退一赔十。保留小票→12315投诉→起诉。
租房合同没到期房东赶人怎么办？
房东违约。可要求继续履行或赔偿损失（第577条、第584条）。
遗嘱怎么立才有效？
自书遗嘱：亲笔书写+签名+年月日。公证遗嘱效力不再优先（第1142条：以最后的遗嘱为准）。
被网络暴力/造谣了怎么办？
侵犯名誉权/隐私权（第1024条、第1032条）。可要求平台删帖（第1195条），起诉赔偿。
诉讼时效过了还能起诉吗？
可以起诉，但对方提时效抗辩则丧失胜诉权（第192条、第193条）。一般时效3年（第188条）。
婚前财产离婚时怎么分？
婚前个人财产归个人（第1063条）。婚后增值部分可能为共同财产。
📚 法律小课堂✕

📚 法律小课堂

懂一点法理，少踩一些坑

1. 特别法优于一般法

通俗解释：同一件事，如果有专门的法律规定，优先按专门法律来，民法典只起兜底作用。

生活例子：买到假货，消费者权益保护法规定"退一赔三"（消法第55条），这比民法典的一般赔偿规定更有利于消费者，优先适用消法。但如果消法没规定的事（如精神损害赔偿），再回到民法典。

涉及法律：《立法法》第92条

2. 上位法优于下位法

通俗解释：法律的等级从高到低是：宪法 > 法律（民法典）> 行政法规 > 地方性法规 > 规章。上级规定优先，下级规定不能和上级冲突。

生活例子：某地方法规规定借条利息最高年24%，但国务院规定民间借贷利率上限为LPR四倍（约14.8%）。地方法规与行政法规冲突时，以行政法规为准。如果行政法规和民法典冲突，则以民法典为准。

涉及法律：《立法法》第88-89条

3. 新法优于旧法

通俗解释：同一件事，前后有两部同级别的法律规定不一样，以后出的为准。

生活例子：原来的《合同法》规定某些合同必须书面形式，而民法典第469条放宽为"书面形式、口头形式或者其他形式"。现在签合同口头约定也有效，以民法典的新规定为准。

涉及法律：《立法法》第92条

4. 法条竞合

通俗解释：一个行为同时触犯了多条法律，但只能选一条来用。一般选最符合行为本质、保护最充分的那条。

生活例子：房东把有安全隐患的房子租出去导致租客受伤，这既属于侵权（民法典第1165条），也属于违约（租赁合同义务第708条）。租客可以选——主张违约赔期待利益，或主张侵权赔实际损失+精神损害。这就是"请求权竞合"，由当事人自己选最有利的。

涉及法律：民法典第186条（违约责任与侵权责任竞合时择一主张）

5. 诉讼时效中断 vs 中止

通俗解释：

• 中断：你去讨债 / 对方认账 / 你起诉了 → 时效重新算3年（清零重启）
• 中止：发生你挡不住的事（地震 / 重病 / 被拘禁）→ 时效暂停，事过了接着算

生活例子：朋友2022年1月借了5万一直没还。你2024年6月去要，他口头说"下个月还"——这构成承认债务，时效从2024年6月重新起算3年，到2027年6月才到期。如果你在2024年12月被车撞昏迷了3个月，这3个月不算时效，从2025年3月醒了再接着算。

涉及法律：民法典第194条（中止）、第195条（中断）

⚖ 法律援助申请指引

什么是法律援助
法律援助是国家建立的为经济困难公民和符合法定条件的其他当事人无偿提供法律咨询、代理、刑事辩护等法律服务的制度（《法律援助法》第二条）。
申请条件
同时满足以下三项：
1. 申请事项属于法律援助范围（见下方）
2. 有证据证明为维护自己合法权益需要法律帮助
3. 家庭经济困难，无力支付法律服务费用。标准：申请人及其共同生活家庭成员月人均收入不高于当地最低工资标准（部分地区按低收入标准执行）
哪些事项可以申请（民事/行政）
• 依法请求国家赔偿
• 请求给予社会保险待遇或社会救助
• 请求发给抚恤金
• 请求给付赡养费、抚养费、扶养费
• 请求确认劳动关系或支付劳动报酬
• 请求工伤事故、交通事故、医疗事故、食品药品安全事故人身损害赔偿
• 请求环境污染、生态破坏损害赔偿
• 因见义勇为产生民事权益
• 因家庭暴力、虐待、遗弃主张权利
• 残疾人、老年人、未成年人、妇女请求人身损害赔偿
• 因土地承包经营流转等合法权益受侵害
哪些人可以免予经济困难审查
• 无固定生活来源的未成年人、老年人、残疾人
• 社会救助/司法救助/优抚对象
• 申请支付劳动报酬或工伤赔偿的进城务工人员
• 刑事案件中有盲聋哑、精神病人、可能无期/死刑等情形的
需要准备的材料
1. 法律援助申请表（可口头申请由工作人员代填）
2. 居民身份证或其他有效身份证明
3. 经济困难状况说明及诚信承诺书（免审情形除外）
4. 与申请事项相关的案件材料
申请流程
携带材料 → 到法律援助机构窗口提交 → 7日内审查作出决定 → 决定给予的，3日内指派律师 → 律师提供免费法律服务
